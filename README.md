# VoyageCore

VoyageCore is a C++ Travel and Tour Management System developed using Object-Oriented Programming (OOP) concepts. The system allows management of tour packages, customers, and bookings through a menu-driven interface.

## Features

* Add, update, search, and delete tour packages
* Manage customer records
* Create and manage bookings
* Booking cancellation and refund processing
* Seat availability tracking
* Low seat alerts and reporting
* File handling for persistent data storage

## Technologies Used

* C++
* Object-Oriented Programming (OOP)
* File Handling
* Visual Studio Code
* Git & GitHub

## How to Run

Compile the project:

g++ VoyageCore.cpp -o VoyageCore

Run the program:

./VoyageCore

## Concepts Demonstrated

* Classes and Objects
* Encapsulation
* Constructors
* Operator Overloading
* File Handling
* Menu-Driven Programming
